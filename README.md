## Conventions

- 각 카테고리는 directory로 구분하며 directory 명은 소문자로 작성한다.
- 컨텐츠 파일명에는 정렬을 위한 index prefix를 삽입한다.
  - example 
	  - 0_getting_started.md
  	- 1_examples.md
- 파일 확장자는 md로 통일한다.
- 문서 작성 후 필요한 경우, TOC 수정한다.
- 불필요한 commit은 하지 않는다.
- commit 메시지는 제목과 간략한 내용으로 작성한다.
